Commit Email Bot

commit-email-bot sends emails when changes are pushed to a Git repository on Github.

Getting Started

Install the commit-emails GitHub app and give it access to the repositories you want notifications for.

In your repo, commit a file called .github/commit-emails.json that specifies the recipients:

{
  "mailingList": "alice@example.com,bob@example.net"
}

Every email from commit-email-bot contains the string jD27HVpTX3tELRBjcpGsK6io7 followed by the name of the repo. You can use this to easily filter commit notifications in Gmail.
